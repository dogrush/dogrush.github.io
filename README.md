# Jianlei Yang's Homepage

Personal academic website of [Jianlei Yang (杨建磊)](https://dogrush.github.io/), Professor at School of Computer Science and Engineering, Beihang University.

## Site Overview

- **Research**: Intelligent computing systems, ML/LLM systems, and embodied intelligence
- **Publications**: 33 journal papers and 50 conference papers
- **Code**: Open-source projects including [CIMFlow](https://github.com/BUAA-CI-LAB/CIMFlow)
- **Teaching**: Courses on intelligent computing architectures

## Local Development

```bash
bundle install
bundle exec jekyll serve -l -H localhost
```

The site will be available at `http://localhost:4000`. Changes to Markdown files are auto-reloaded; changes to `_config.yml` require restarting Jekyll.
